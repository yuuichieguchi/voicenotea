Voicenotea について

Voicenotea は、ユーザーが音声でメモを素早く作成できるシンプルなメモアプリです。本プライバシーポリシーは、当アプリがどのようにあなたの情報を取り扱うかについて説明しています。

収集データ

音声入力

  • RECORD_AUDIO 権限の使用：アプリはデバイスのマイクから音声を取得します
  • この音声データは、デバイス上の音声認識機能を使用して、テキストに変換されます
  • 音声データはデバイス内でのみ処理され、外部サーバーに送信されません

メモデータ

  • 作成・編集したメモはすべてデバイスの内部ストレージに保存されます
  • ユーザーは自由にメモを削除できます

その他のデータ

Voicenotea は以下のデータを収集・送信しません：

  • 個人識別情報（名前、メールアドレスなど）
  • 位置情報
  • デバイス識別情報
  • 使用統計
  • アナリティクスデータ

データの保護とセキュリティ

  • ローカルストレージのみ：すべてのデータはデバイス内でのみ保存されます
  • インターネット送信なし：メモや音声データは一切インターネットに送信されません
  • プライバシー優先：ユーザーのプライバシーは最優先です

権限について

RECORD_AUDIO 権限

  • メモを音声で作成するために必要です
  • マイクへのアクセスはアプリ使用時のみ

データの削除

  • メモはいつでもアプリ内から削除できます
  • アプリをアンインストールすると、すべてのデータが削除されます

このプライバシーポリシーの変更

Voicenotea は、本プライバシーポリシーを予告なく変更することがあります。変更があった場合は、このページで通知します。
