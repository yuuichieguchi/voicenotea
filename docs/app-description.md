# Voicenotea - 詳しい説明

## 【概要】

話すだけでメモが完成。Voicenotea は、Android の音声認識機能を使って、素早くシンプルにメモを作成できるアプリです。タイピングの手間を省き、思いついたことをすぐに記録できます。

---

## 【主な機能】

### 1. 音声入力でメモを素早く作成
- マイクボタンをタップして話しかけるだけで、音声がリアルタイムでテキストに変換されます
- 複数の文を連続して認識し、自動的に保存されます
- タイピングが不要なので、運転中や手がふさがっているときにも便利です

### 2. 継続音声認識で複数の文を自動保存
- 話し終わった後、自動的に次の音声入力に切り替わります
- 一度の操作で複数の文を記録できるため、効率的にメモが作成できます

### 3. メモの編集と削除
- 作成したメモはいつでも編集できます
- タイトルと本文を自由に修正・追加可能
- 不要なメモは簡単に削除できます

### 4. 複数選択で一括削除
- メモを長押しして複数選択できます
- 選択したメモを一括削除でき、管理が簡単です

### 5. シンプルで使いやすいUI
- 余分な機能はなく、必要な機能に絞ったシンプル設計
- 初めてのユーザーでも直感的に操作できます
- ダーク（濃色）テーマで目に優しい表示

---

## 【こんなときに便利】

✓ 移動中に思いついたアイデアを記録したい
✓ 会議や授業の内容をメモしたい
✓ 日記や日々の出来事を記録したい
✓ タイピングが面倒なときに素早くメモしたい
✓ 手が離せないときに音声で記録したい

---

## 【プライバシーについて】

- **データはデバイス内に保存**：すべてのメモはあなたのデバイス内にのみ保存されます
- **インターネット送信なし**：メモデータや音声は一切外部に送信されません
- **個人情報不要**：サインアップやログインは不要です

---

## 【利用条件】

- Android 7.0（API レベル 24）以上
- マイク機能が必要です
- インターネット接続は不要です
