HOW TO BUILD & RUN
================================================================================

1. Open in Android Studio:
   open -a "Android Studio" /Users/eguchiyuuichi/projects/voicenotea

2. Let Gradle sync (File → Sync Now)
   First build: ~2-3 minutes

3. Create Android Virtual Device (API 24+)
   Tools → Device Manager → Create Device

4. Run:
   - Click Run button (green play icon)
   - Or: Shift+F10 (Mac: Cmd+R)
   - Grant microphone permission when prompted

5. Test:
   - Tap Mic button to start recording
   - Speak something
   - Tap Stop button
   - Watch it transcribe
   - Memo appears in list
   - Tap to edit or delete
